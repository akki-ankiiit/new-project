# The Golf Mill — Premium Golf Simulator Website

## 1. Project Description
The Golf Mill is a premium indoor golf simulator training center offering state-of-the-art simulators, precision coaching, and access to 200+ courses. The website targets serious golfers and enthusiasts looking to upgrade their game with professional training tools. Core value: luxury experience, credibility, and easy booking.

## 2. Page Structure
- `/` — Home (hero, value, CTA strip, FAQ, testimonials, final CTA)
- `/services` — Services (simulator packages, coaching, events)
- `/contact` — Contact (form, map, info)
- `*` — 404 Not Found

## 3. Core Features
- [x] Glass-morphism luxury UI (Dark theme)
- [x] Hero section with booking inquiry form
- [x] Services showcase page
- [x] Contact page with form + map
- [x] FAQ accordion
- [x] Testimonials section
